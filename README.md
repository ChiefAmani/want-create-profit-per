# Residential Cleaning Profitability Calculator

A premium, automated Excel calculator designed for residential cleaning business owners (Window, Gutter, Siding, and Pressure Washing). This tool is built to be sold as a digital product, offering advanced financial insights that help owner-operators stop undercharging and start scaling.

## Value Proposition
Most cleaning business owners guess their prices. This calculator provides mathematical certainty. It tracks labor, chemical costs, drive time, and equipment wear-and-tear to generate exact profit margins. 

### Key Features
- **Price vs. Labor Breakdown**: Exact cost of employee time including taxes and drive time.
- **Chemical Cost Tracking**: Micro-costing for SH, surfactants, and degreasers.
- **Callback Cost Impact**: Calculates the devastating margin impact of returning to a job site.
- **Break-Even Calculator**: Determines exactly how many jobs are needed to cover monthly overhead.
- **"Are You Undercharging?" Indicator**: Visual gauge that flags low-margin jobs.
- **Good/Better/Best Pricing Generator**: Automatically creates 3-tier pricing to increase average ticket size.

## Developer Instructions
This project uses a Python script to programmatically generate the `.xlsx` file. This ensures the Excel file is perfectly formatted, formulas are locked, and it can be version-controlled.

### Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the generator: `python generate_calculator.py`
3. The output will be `Cleaning_Profit_Calculator.xlsx`.
